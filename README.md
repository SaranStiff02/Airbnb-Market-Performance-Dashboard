## 📌 Project Overview

This project presents an Airbnb market analysis dashboard built using Tableau.

The dashboard analyses Airbnb listings by neighbourhood, room type, average price, review distribution, estimated revenue, and host performance. It is designed to help understand listing demand, pricing patterns, review quality, and revenue opportunities.

---

## 🎯 Business Questions

- Which neighbourhoods have the highest listing concentration?
- How does average price vary by room type and neighbourhood?
- Which room types receive the highest number of reviews?
- Which listings generate the highest estimated revenue?
- Which hosts generate the most estimated revenue?
- How do room type and quarter filters affect the market view?

---

## 📊 Dashboard Features

- 🏘️ Neighbourhood distribution by room type
- 💰 Average price comparison
- ⭐ Popularity and quality rating by room type
- 🧾 Review distribution by room type
- 🏆 Top 10 listings by estimated revenue
- 👤 Estimated revenue by host name
- 🎛️ Interactive filters for room type and quarter

## 🔍 Key Insights

- Entire home/apartment listings appear to dominate several neighbourhoods.
- Average price varies significantly across neighbourhoods.
- Some hosts generate much higher estimated revenue than others.
- Review distribution helps identify popular room types.
- Top revenue listings show strong earning potential in selected locations.

---

## 💼 Business Use Case

This dashboard can help Airbnb hosts, property investors, and short-term rental analysts understand market performance.

It supports decisions related to pricing strategy, neighbourhood selection, room type performance, host benchmarking, and revenue opportunity analysis.
